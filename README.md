tik
====

## install

`npm install -g tik`

## usage

`tik <key> [newvalue]`

retrieve `key` value if no `newvalue` specified, or set the value of `key` to `newvalue` if it is.

## additional commands

#### tik ls

list all keys and values

#### tik lskeys

list all keys

#### tik rm <key>

remove `key` from db
